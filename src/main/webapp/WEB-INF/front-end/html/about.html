关于我们

    熊熊网是国内领先的计算教育服务平台。独创的互动式教学模式，让学生学习编程的过程更有趣，更高效。 我们笃定计算教育在未来将会是一种通识教育。在衣食住行都被网络化的如今，人机之间的交流时间甚至要远远多于人与人的交流时间。我们日常所浏览的所有网站、使用的软件乃至操作系统，都是由程序编写而成，编程作为唯一能使人与计算机进行交流的工具，已经变得越来越重要，而学习编程的人也日渐增多。就连美国总统奥巴马都曾在视频中呼吁：美国的孩子除了玩游戏，还应该具备如何做游戏的能力。 由于编程的复杂性和专业性，导致许多学习编程的人对此都大感头疼，编程似乎成为了"程序猿"的独家技能，而对于绝大部分的人来说只能望而却步。我们就是为了解决这个难题而来，熊熊首先就是要让大家感受到编程的乐趣，消除编程复杂难懂的印象：在熊熊学习，用户无需配置编程环境，所有学习和操作在网页中均能轻松实现；"跟随式"的教学过程如同老师手把手教学，消除了用户学习过程中的干扰因素，让编程的学习一气呵成；同时，网站还将理论与实践融为一体，用户能够一边学习一边进行实际操作，掌握知识点的同时便能实践，从而加深了学习记忆。通过这种最纯粹的人机交互，就算是最零基础的"小白"用户也能轻松搞定。 "少即是多"是熊熊团队一直坚持的产品哲学之一。熊熊通过"课程X"让用户来决定下一门开设的课程是什么。"在教育领域各式各样的课程太多，反而会提高用户的选择成本，因此我们不求过多，只提供最好的。对于熊熊来说，每上线一门课程，就势必要做到精品中的精品。" 熊熊还有一项国内编程教育首创的功能："时光机"。这项功能类似编程过程中的debug：时光机让用户能够随时回放之前的编程步骤，并且以清晰的图形形式一步步进行显示，充分梳理程序运行中的脉络。用户在学习过程中及时发现自己错误的地方并加以改正，从而养成正确的编程习惯。（时光机引用了开源项目PythonTutor (http://pythontutor.com/)，感谢 Philip Guo的杰出工作成就） 熊熊团队是一帮充满活力的90后年轻人。"熊熊"两字对于我们有着非同寻常的一番含义：网站叫做熊熊，一是因为觉得学习写代码的过程完全没必要那么枯燥乏味，只需一台电脑一杯热茶，抱着萌萌哒的心情也可以学好编程。二是取"猛犸"的谐音，猛犸象能够在漫长而残酷的寒冷气候中生存，而这一份坚韧也是创业者最重要的精神力量。不论过去经过多少风雨，多少误解，熊熊都将不忘初心。相信通过不断的努力和改进，熊熊将会得到越来越多用户的认同。 熊熊团队深知我们的成长离不开大家的关心和厚爱，一路走来，之后用户的不断支持才让我们有激情和力量走下去。我们不忘初心，永远会以一颗谦虚的心为中国乃至世界的教育贡献自己的绵薄之力。
